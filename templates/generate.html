Log Data Overview
Log Sources:
  • AuthService - Authentication events
  • PaymentService - Payment processing
  • DatabaseService - Database operations
  • NotificationService - Notifications
  • CacheService - Cache operations
Log Levels:
  • INFO (70%) - General information
  • WARN (10%) - Warning messages
  • ERROR (10%) - Error messages
  • DEBUG (10%) - Debug information
Data Features:
  • Realistic timestamps (last 365 days)
  • Error spikes monthly
  • Dynamic message templates
  • Fake user data and IP addresses
Kibana Integration

When you enable Kibana integration, the following will be created:

Data View:
  • Index pattern: unstructured-logs
  • Time field: @timestamp
Discover Sessions:
  • Retrieve All Logs
  • Count Logs by Level
  • Count Logs by Source
  • Retrieve Error Logs
  • Parse AuthService (DISSECT)
  • Parse AuthService (GROK)
These sessions demonstrate ES|QL query capabilities including DISSECT and GROK parsing.
