Configuration Help
Elasticsearch Settings:
  • Default port is usually 9200
  • Use HTTPS if your cluster requires it
  • API keys can be used instead of username/password
Kibana Settings:
  • Default port is usually 5601
  • Must have access to create saved objects
  • Same credentials as Elasticsearch typically work
Security Tips:
  • Change default passwords in production
  • Use environment variables for credentials
  • Enable HTTPS for external connections
